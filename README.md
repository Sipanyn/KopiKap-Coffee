# KopiKap Coffee ☕️

A sleek and modern coffee shop website built with Vite.

## 🚀 Features
- Fast and lightweight Vite setup
- Responsive design
- Easy deployment with Vercel

## 🛠️ Tech Stack
- Vite
- HTML/CSS/JavaScript
- TailwindCss
- Redux ToolKit
- React Router
- React Query
- GitHub + Vercel

## 📦 Installation 
```bash
git clone https://github.com/Sipanyn/KopiKap-Coffee.git
cd KopiKap-Coffee
npm install
npm run dev

# KopiKap Coffee ☕️

A sleek and modern coffee shop website built with Vite.

## 🚀 Features
- Fast and lightweight Vite setup
- Responsive design
- Easy deployment with Vercel

## 🛠️ Tech Stack
- Vite
- HTML/CSS/JavaScript
- TailwindCss
- Redux ToolKit
- React Router
- React Query
- GitHub + Vercel
-
